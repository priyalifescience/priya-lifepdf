# Priya LifePDF — Pharma Notice

This tool was built specifically for pharmaceutical, biotech, and MedTech 
professionals in Ireland.

## What this tool does NOT do
- It does not upload your files to any server
- It does not log file names, contents, or metadata
- It does not use cookies for tracking
- It does not require registration or login

## Regulatory context
Pharmaceutical documents (batch records, validation protocols, SOPs, 
regulatory submissions) are controlled documents under EU GMP and HPRA 
guidelines. This tool was designed to allow processing of such documents 
without creating an unauthorised copy on a third-party server.
